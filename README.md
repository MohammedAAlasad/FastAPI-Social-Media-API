# FastAPI Social Media API

This project is a REST API built with FastAPI that simulates the backend of a simple social media application. It allows users to create accounts, authenticate using JWT tokens, manage posts, and upload images or videos.

The project follows modern backend development practices by using asynchronous programming with FastAPI and SQLAlchemy, database migrations with Alembic, and request validation with Pydantic. It was developed as a learning project to gain hands-on experience building scalable and maintainable APIs.

Technologies

* FastAPI
* SQLAlchemy
* Alembic
* Pydantic
* JWT Authentication
* Python Multipart
* Uvicorn

This project demonstrates the fundamentals of building a secure REST API, including authentication, CRUD operations, database relationships, file handling, and API documentation through Swagger UI.
